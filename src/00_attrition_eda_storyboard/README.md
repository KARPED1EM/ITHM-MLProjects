# AttriPredict - Attrition EDA Storyboard

用 PPT 式结构交替穿插 Markdown 与可视化，覆盖以下内容：

- 数据健康、类别不平衡、人口画像、工作模式、激励反馈等多条特征路线的洞察。
- 与 01 `brute-force_model_search`、02 `lr_production_playbook` 共用的数据口径与特征假设，方便把 EDA 结论直接映射到特征工程、SMOTE 组合和模型选择。
- 额外的部门/岗位、晋升停滞、满意度矩阵等图表，帮助解释为何后续要尝试 log1p、StandardScaler、交互特征、以及多种过采样器。
